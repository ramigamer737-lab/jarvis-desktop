Capabilities

Everything you need,
built right in

JARVIS ships with 52 tools, real LLM integration, and a self-improvement engine that writes new tools automatically.

🤖
ReAct AI Agent
52 built-in tools with a Reason+Act loop. Connects to OpenAI, Anthropic, Ollama, or runs in mock mode.
52 tools · 4 LLM providers
🔊
Voice Interface
Wake word detection, real-time speech-to-text, and natural TTS responses. Talk to JARVIS hands-free.
Wake word · STT · TTS
🖥️
PC Control
File management, app launching, system monitoring, clipboard control, screenshot capture via natural language.
File · App · System · Clipboard
🧠
Self-Improvement
JARVIS generates, tests, and registers new tools automatically using a 7-step improvement cycle.
7-step cycle · Hot-reload
⚡
FastAPI Backend
Full REST API on localhost:8000 with OpenAPI docs. Every feature accessible via HTTP.
REST · OpenAPI · localhost:8000
🔄
Auto-Updates
Background update checks via GitHub Releases. Downloads silently, then prompts you to restart.
electron-updater · GitHub Releases
Installation

Up and running in 4 steps

No command line required. Download, run the installer, and JARVIS is ready.

01
⬇️
Download
Click the download button above to get the JARVIS installer.
02
🚀
Run Installer
Run JARVIS_Setup_v1.0.0.exe — the Modern UI wizard guides you through.
03
⚙️
Configure
Choose your install directory, optional components (service, auto-start), and desktop shortcut.
04
✨
Launch
Open JARVIS from the Start Menu. The dashboard loads instantly — backend starts automatically.
